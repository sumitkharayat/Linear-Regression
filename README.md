# Linear-Regression on Cryptocurrency Dataset
Performing the linear regression on the cryptocurrency market is known for its high volatility and rapid pace. This dataset provides a real-time snapshot of the top 250 cryptocurrencies by market capitalisation 


This project applies Linear Regression on a cryptocurrency dataset to analyze the relationship between trading volume and price.

## Dataset
An open-source cryptocurrency dataset dated 2025-12-22 


## Objective
To predict cryptocurrency price based on trading volume using Linear Regression.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Result
The regression line shows how price varies with trading volume.
